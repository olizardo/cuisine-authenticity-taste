# Plan - Re-engineer Cuisine Analysis to Match the Comprehensive Geometric GDA Pipeline

This plan outlines the re-engineering of the entire cuisines analysis and report to mirror the exact sequence, rigor, and depth of the `analysis-aesthetic-PCA.qmd` pipeline from the `childress-lizardo-aesthetic-politics` project, but utilizing **Multiple Factor Analysis (MFA)** to properly group our continuous active variables by their survey blocks.

## Analytical & Reporting Sequence to Implement

We will construct `statistical-findings-report.qmd` with the following comprehensive, step-by-step sections:

### 1. Setup, Data Hygiene, & Descriptive Table
- **Data Loading**: Load and clean the dataset, ensuring the 12 derived/analysis-product variables are eliminated from the raw data and computed on the fly.
- **Table 1: Descriptive Statistics**: Use `gtsummary::tbl_summary` to produce a publication-quality descriptive table containing means, standard deviations, ranges, and frequencies for all active and supplementary variables, saved as `Tabs/desc.html`.

### 2. Exploratory Bivariate Correlations
- **Pairwise Correlations**: Compute a complete pairwise Pearson correlation matrix for all continuous active and supplementary variables.
- **Plot 1: Corrplot**: Save a high-resolution, publication-ready correlation matrix heatmap as `Plots/corr.png` using `corrplot` with diverging colors (`RdYlBu`), showing coefficients and lower-triangle values.

### 3. Multiple Factor Analysis (MFA)
- **MFA Execution**: Run `MFA` grouping active variables into survey blocks (`dine_out`, `food_qual`, `food`, `authenticity` treated as continuous) and defining supplementary variables (`quant.sup` and `quali.sup`).
- **Plot 2: Scree Plot**: Generate a publication-quality scree plot of eigenvalues (`Plots/eig.png`) showing variance explained.
- **Plot 3a & 3b: Variable Correlations with Dimensions**: Generate two clean, segmented correlation heatmaps (`Plots/mfa_varcor_1.png` and `Plots/mfa_varcor_2.png`) showing the correlation of active variables with the first three dimensions.

### 4. Multivariate Ideological Regressions (OLS)
- **OLS Regression Models**: Fit three OLS models predicting coordinates for `Dim.1`, `Dim.2`, and `Dim.3` using all socio-demographic covariates and social and economic ideologies.
- **Plot 4: Predicted Ideological Scores**: Map and plot the predicted marginal effects of social ideology across the three dimensions with 95% confidence intervals, saving it as `Plots/polpred.png` (using the three-panel horizontal facet structure).

### 5. Formal Multivariate Analysis of Variance (MANOVA)
- **MANOVA Model**: Fit a formal MANOVA on the first three MFA dimensions using Pillai's trace to test for centroid differences across all 12 categorical/demographic predictors.
- **Table 2: HTML MANOVA Results**: Format a clean, publication-ready MANOVA results table as an HTML file in `Tabs/mfa-manova-tab.html` using the exact layout of Table 4 in the reference file.

### 6. Post-Hoc Multivariate Diagnostics & Centroid Projections
- **Ellipse-Based Projections**: Compute 95% standard error confidence ellipses for group centroids.
- **Plots 5, 6, & 7: 2D Pairwise Projections**: Plot 2D projections (Dim 1 vs. Dim 2, Dim 1 vs. Dim 3, Dim 2 vs. Dim 3) with standard error ellipses for:
  - Social Ideology (`Plots/mfa-pairwise-projections.png`)
  - Education (`Plots/mfa-pairwise-educ-projections.png`)
  - Religiosity (`Plots/mfa-pairwise-relig-projections.png`)

### 7. Hierarchical Clustering on Principal Components (HCPC)
- **HCPC Pipeline**: Perform hierarchical clustering on the first three dimensions using Ward's minimum variance criterion and k-means consolidation to identify 4 distinct, theoretically grounded "taste subcultures".
- **Table 3: Subculture Centroids**: Generate an HTML table of cluster centroids and proportions, saved as `Tabs/hcpc-clusters-tab.html`.
- **Plot 8: Subculture Map**: Generate a 2D scatter plot (`Plots/mfa-hcpc-clusters.png`) of individual coordinates colored by cluster with normal-distribution ellipses and cluster centroids.

### 8. Socio-Demographic Predictors of Subculture Membership (Multinomial Logistic Regression)
- **Multinomial Model**: Fit `nnet::multinom` predicting cluster membership using continuous quadratic terms ($\text{Age}^2$ and $\text{Income}^2$) and reference category "Everyday Convenience Seekers" (or another reference group representing baseline taste).
- **Table 4: Regression Output**: Save a publication-ready regression table as `Tabs/mfa-multinom-tab.html` using `sjPlot::tab_model`.
- **Predictive Probability Visualizations**:
  - **Plot 9: Key Predictors Facet**: A 2x2 faceted plot of predicted probabilities with 95% CIs across Social Ideology, Religiosity, and Childhood Arts (`Plots/mfa-multinom-key-probabilities.png`).
  - **Plot 10: Demographic Predictors Facet**: A 2x2 faceted plot of predicted probabilities across Race, City Size, and Education (`Plots/mfa-multinom-demographics-probabilities.png`).
  - **Plot 11: Continuous Age Curves**: Continuous lifecycle predicted probability curves across age with 95% shaded confidence ribbons (`Plots/mfa-multinom-age-curves.png`).
