# Plan: Create Structured Codebook for qualtrics.dta

We will create a structured codebook in Quarto Markdown format (`qualtrics-data-codebook.qmd`) in the root directory. This codebook will serve as a comprehensive reference manual for the raw/complete survey dataset `dat/qualtrics.dta` (and its identical counterpart `genresobjects.dta`), matching the style and depth of `prolific-data-codebook.qmd` but fully adapted to the 709 variables present in the Qualtrics data.

### 2. Introduction & Dataset Overview
- **Dataset Size:** 1,821 observations (rows) and 709 variables (columns).
- **Comparison to Prolific Dataset:** Clarify that this is the full/raw Qualtrics survey dataset, whereas `cultdat.dta` (from `prolific-data-codebook.qmd`) is a smaller cleaned subset (1,258 rows and 168 columns).
- **Identical Files:** Explicitly document that `dat/qualtrics.dta` and `dat/genresobjects.dta` are 100% identical.

### 3. Data Loading Reference
Provide copy-pasteable R and Python snippets to load the dataset:
- **R:** `read_dta("dat/qualtrics.dta")` using the `haven` package.
- **Python:** `pd.read_stata("dat/qualtrics.dta")` using `pandas`.

### 4. Global Variables Directory Table (Categorized)
Since listing all 709 variables in a single flat table is overwhelming and impractical, we will present a **structured summary directory table** grouping variables by survey blocks and prefixes:
- **Survey Metadata:** `startdate` to `userlanguage` (11 columns)
- **Early Demographics:** `sex`, `age`, `race`, `in_school`, `educ`, `income` (8 columns)
- **Movie Genres:** `hours_movies`, `like_movies_genres_11` to `_23` (13 columns)
- **Music Genres:** `hours_music`, `like_music_genres_21` to `_40` (20 columns)
- **TV Genres:** `hours_tv`, `like_tv_genres_11` to `_25` (15 columns)
- **Individual Movie Preferences:** `like_movies_125` to `_211` (37 columns, including attention checks)
- **Individual Music Preferences:** `like_music_132` to `_222` (35 columns)
- **Individual TV Preferences:** `like_tv_107` to `_192` (35 columns)
- **Music Player Experimental Blocks (iTunes, Winamp, Spotify):**
  - iTunes/Winamp/Spotify sorting and playlist variables (approx. 180 columns)
- **TV Practice Block:** `tv_practice_107` to `_192` (35 columns)
- **Leisure Preferences & Practices:** `like_leisure` and `practice_leisure` (29 columns)
- **Sports Preferences & Practices:** `like_sports` and `practice_sports` (29 columns)
- **Cosmopolitanism Block:** `cosmo_1_1` to `cosmo_3_7` (24 columns)
- **Demographic Block:** political stances, occupational networks (`network_1` to `_30`), parental background, arts engagement (approx. 43 columns)
- **Analytical & Recoded Variables:** Stata/R derivative columns like dummies, indices, and MCA coordinates appended at the end of the dataset (approx. 150 columns)

### 5. Detailed Survey Blocks & Variable Specifications
For each survey block, we will document:
- The exact question wording from `qualtrics.pdf`.
- The response scale options and their numeric values (e.g., 1–7 scale for liking, 1–4 scale for practice).
- Complete lists of items with their variable names and survey text labels.
- Attention-check questions (e.g., `like_movies_210` = "Click 'Very Much Dislike'", `like_movies_211` = "Click 'Mostly Like'").
